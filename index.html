Form Penilaian Karyawan

Di bawah ini tersedia lembar penilaian KPI untuk seluruh karyawan warehouse. Setiap karyawan memiliki satu halaman form tersendiri dengan format tabel harian seperti contoh yang kamu kirim.

Task standar = 1 poin Tanggung Jawab Pekerjaan = 2 poin Isi angka pada tabel untuk update leaderboard
Secara default halaman berada di mode lihat. Hanya user yang masuk lewat akses editor yang bisa mengubah angka pada tabel, sedangkan user lain hanya bisa melihat hasil penilaian dan leaderboard.
Saat di-deploy ke Vercel, tombol cloud akan membaca dan menulis data JSON user aktif ke Vercel Blob melalui endpoint API project ini.
Tampilan ini dibuat sebagai halaman statis dan bisa langsung dibuka dari file index.html.
